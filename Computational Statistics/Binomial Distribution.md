Number of times success happens in N tries in [[Bernoulli Distribution]].Ex. Number of heads in N tosses. 

When $X\sim B(\pi,N)$, X can take the values  $$X\in\{0,1,...N\}$$ The Probability Distributions is:$$ P(X=k) = \binom{N}{k}\pi^k(1-\pi)^{N-k} $$ 
The Expected Value is $$\mathbb E[X]=N\pi$$ and Variance is $$Var[X]=N\pi(1-\pi)$$

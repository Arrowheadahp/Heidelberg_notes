$$ Var[X] = E[(X-E[X])^2]$$
- $Var[X] = E[X^2]-E[X]^2$
- $Var(aX+b) = a^2Var(X)$
#### Properties
- ==$Cov[X,Y] = E[XY] - E[X]E[Y]$==
- $Var(X+Y) = Var(X) + Var(Y) + 2Cov[X,Y]$
- If X, Y are independent, $Cov[X,Y]=0$
- $Var[\sum X_i] = \sum Var[X_i]$ if $X_i$ are independent
- $Cov[aX+b, cx+d] = acCov[X,Y]$
- $Cov[X,Y] = Cov[Y,X]$

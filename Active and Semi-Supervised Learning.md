This is when the model is given some labelled data as in Semi-Supervised Learning, and some unlabelled data. The unlabelled data is predicted on and the ones that the model is not sure about (near the decision boundary) is given to a human for labelling. Then it becomes labelled data to be trained upon. Sometimes random observations are also asked to be labelled for testing purposes.

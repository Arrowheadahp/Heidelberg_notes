2 types of [[Machine Learning]] models:

1. Unstructured Machine Learning: 
   Where input is an image and text and output is a real number.
	1. [[Classification]]
	2. [[Regression]]
2. Structured Machine Learning:
   Where input is image or text and output is a complex structured object like labelling or image generation

3 classes of [[Machine Learning]] problems

1. [[Supervised Learning]]: Fully labelled annotated Dataset
2. [[Unsupervised Learning]]: No annotations (ex: Clustering)
3. [[Reinforcement Learning]]: Reward for correct action

Lots of variants of [[Machine Learning]] problems

1. On-line learning
2. Continual Learning
3. Self-Supervised Learning
4. [[Active and Semi-Supervised Learning]]
5. [[Pretext Task]]: Train for problem A, fine-tune for problem B
